# Copy Frameworks — Referência Completa

## AIDA
```
Attention: Hook visual + afirmação de impacto (para o scroll)
Interest:  Contexto de dor que o público reconhece
Desire:    Solução com número específico de resultado
Action:    CTA único, claro e com baixa fricção
```

## PAS
```
Problem:   "Processo sem padrão = retrabalho eterno" (1 frase)
Agitation: "R$X por mês em ineficiência que ninguém está medindo"
Solution:  "DMAIC em 4 semanas, −30% custo"
```

## Storytelling / Before-After-Bridge
```
Before:  Cena de caos real — o antes da operação
After:   Estado ideal após transformação com número
Bridge:  O método ou ferramenta que gerou a mudança
CTA:     Como o leitor replica
```

## Direct Response
```
Hook:   Benefício direto ou número chocante
Prova:  Resultado real, screenshot ou dado verificável
Oferta: O que está sendo entregue especificamente
CTA:    Ação única — "Chame no WhatsApp"
```

## Viral Hook
```
Afirmação contra-intuitiva ou controversa
"X% das empresas cometem esse erro sem saber"
"Você acha que tem processo. Mas tem caos operacional."
"O retrabalho não é culpa da equipe — é culpa do processo."
```

## Educational Copy
```
Premissa simples: o que é Lean / automação / Six Sigma em 1 frase
Explicação humanizada: analogia ou exemplo real do nicho
Aplicação prática: o que o leitor faz hoje com esse conhecimento
```

## Authority Copy
```
Posição: número de casos, metodologia, certificação
Prova:   resultado real de cliente com contexto específico
Diferenciação: por que essa abordagem supera as alternativas
```

## Viral Mechanics — Gatilhos

| Gatilho | Aplicação |
|---------|-----------|
| Curiosidade | Hook com lacuna — leitor precisa continuar |
| Identificação | Cena que o público já viveu |
| Contraste | Antes vs depois, certo vs errado |
| Autoridade | Número de casos + metodologia certificada |
| Prova social | Resultado real de cliente com contexto |
| Urgência | Custo de não agir — "cada semana = R$X" |
| Dor | Situação que todos reconhecem mas ninguém nomeia |
