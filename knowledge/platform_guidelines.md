# SmartOps IA — Platform Guidelines

*Propósito: Garante que todo conteúdo esteja corretamente formatado e no tom certo por plataforma antes de publicar.*

---

## Visão Geral

| Plataforma | Tipo de Conteúdo | Tom Primário | Público Esperado |
|---|---|---|---|
| **Instagram** | Carrosséis, Reels, Feed | Didático com prova real | Donos de empresa, gestores |
| **Threads** | Posts curtos de texto | Provocativo, direto, técnico | Profissionais, empreendedores |
| **YouTube** | Shorts e vídeos tutoriais | Educativo, demonstrativo | Buscando resolver problema específico |

## Instagram

### Specs

| Formato | Dimensões | Aspect Ratio |
|---|---|---|
| Feed Post | 1080 × 1080 px | 1:1 |
| Carrossel | 1080 × 1350 px | 4:5 |
| Reels / Story | 1080 × 1920 px | 9:16 |

### Design Rules

- **Background:** Dark (`#06060e` ou `#0d0d1c`) ou cards com border `rgba(255,255,255,.07)`
- **Texto principal:** `#e8e8f0` (off-white)
- **Labels/tags:** `#a78bfa` (roxo claro) para Lean, `#6ee7b7` (verde esmeralda) para Automação
- **Accent:** `#7c3aed` (roxo) para Lean Six Sigma, `#10b981` (verde) para Automação com IA
- **CTA:** `#25d366` (verde WhatsApp) ou botão roxo `#7c3aed`
- **Nunca:** fundo branco puro, gradientes neon, fontes decorativas
- **Border radius:** 12–16px em cards
- **Margem segura:** 48px em todos os lados

### Caption Guidelines

- **Tamanho:** 2–4 frases antes das hashtags
- **Estrutura:** Hook com dor/prova → Método/solução → Resultado específico → CTA → Hashtags
- **Tom:** Profissional mas acessível — como explicando para um cliente no primeiro contato
- **Emojis:** 0–1, só se adicionar contexto visual (✓ para resultado, ⚠ para problema)
- **CTA:** Sempre "Link na bio" ou "Diagnóstico gratuito no link"
- **Nunca:** urgência fake, "aproveite agora", promessas vagas

**Estrutura de caption:**
```
[Hook: caso real ou dor com número]

[Método: o que foi feito em termos simples]

[Resultado: número específico e prazo]

[CTA: diagnóstico gratuito ou link na bio]

#SmartOpsIA #LeanSixSigma #AutomacaoIA [tags de nicho]
```

**Exemplo:**
```
3 meses tentando resolver retrabalho no atendimento. 4 semanas com Lean e o processo ficou padronizado.

Não era falta de esforço da equipe. Era o processo quebrando no mesmo ponto toda vez. DMAIC identifica a causa raiz. Não o sintoma.

Diagnóstico gratuito de 30 minutos — link na bio.

#SmartOpsIA #LeanSixSigma #MelhoriaDeProcessos #PequenasEmpresas #AutomacaoIA
```

### Hashtag Rules

- 4–6 hashtags por post
- Mix: #SmartOpsIA (marca) + #LeanSixSigma (método) + #AutomacaoIA (serviço) + nicho do cliente
- **Lean:** `#LeanSixSigma #DMAIC #MelhoriaDeProcessos #GestaoDeProcessos`
- **Automação:** `#AutomacaoIA #WhatsAppBusiness #AutomacaoDeProcessos #IA`
- **Nicho:** `#PequenasEmpresas #EmpreendedorBR #GestaoEmpresarial #BH`

### Carrossel Rules

- **Slide 1:** Hook forte — dor ou provocação. Fundo escuro, texto grande
- **Slides 2–5:** Um ponto por slide. Dados reais, exemplos concretos
- **Slide final:** CTA + foto ou logo SmartOps IA + "Diagnóstico gratuito"
- **Labels de seção:** pill roxo (`#a78bfa`) para Lean, pill verde (`#6ee7b7`) para Automação
- **Sem:** slides sem hierarquia visual clara, texto em bloco sem respiração

## Threads

### Specs

- Texto puro (imagens opcionais)
- **Máximo:** 500 caracteres
- Sem requisito de dimensão se usar imagem — use 1:1 ou 4:5

### Caption Guidelines

- **Tom:** O mais direto e técnico da marca — pode ser provocativo sem ser agressivo
- **Tamanho:** 2–3 frases curtas e auto-suficientes
- **Hashtags:** 0–1, opcional — nunca começa com hashtag
- **Sem CTA obrigatório** — pode terminar com pergunta técnica, observação seca ou insight

**Formatos que funcionam no Threads:**

1. **Insight técnico curto:**
```
Automatizar processo ruim só faz você errar mais rápido.

Lean primeiro. Automação depois. Essa sequência muda o jogo.
```

2. **Pergunta retórica com dado:**
```
Em quantas empresas o processo só funciona quando o dono está presente?

É padronização, não talento. Com Lean, qualquer pessoa executa igual.
```

3. **Desmistificação:**
```
Lean Six Sigma não é só para indústria.

Funciona especialmente bem em pequenas empresas — porque elas não têm folga para absorver desperdício.
```

4. **Prova rápida:**
```
4 semanas. Processo de atendimento padronizado. Time parou de improvisar.

Isso é DMAIC funcionando em clínica odontológica. Não teoria. Resultado real.
```

### Content Notes

- Nunca soar como anúncio disfarçado de conteúdo
- Engage sem pedir engagement explicitamente
- Números e casos reais sempre que possível
- Pode usar linguagem mais técnica que Instagram (audiência entende)

## YouTube

### Specs

| Formato | Dimensões | Aspect Ratio |
|---|---|---|
| Shorts | 1080 × 1920 px | 9:16 |
| Vídeo Padrão | 1920 × 1080 px | 16:9 |
| Thumbnail | 1280 × 720 px | 16:9 |

### Title Guidelines

- **Tamanho:** 50–65 caracteres (evita truncamento)
- **Estrutura:** Problema ou resultado + como resolver + contexto | SmartOps IA
- **Tom:** Informativo primeiro, persuasivo segundo — YouTube é busca, não feed
- **SEO:** Inclua termos como "Lean Six Sigma", "DMAIC", "automação WhatsApp", "pequena empresa"
- **Nunca:** clickbait vago, títulos sem palavra-chave, CAPS LOCK excessivo

**Exemplos de títulos:**
```
Como Eliminar Retrabalho com Lean Six Sigma em 4 Semanas | SmartOps IA
DMAIC na Prática: Diagnóstico de Processo em Clínica | SmartOps IA
Automação WhatsApp com IA: Como Funciona na Prática | SmartOps IA
8 Desperdícios do Lean — Qual Está Custando Mais na Sua Empresa? | SmartOps IA
```

### Description Guidelines

- **Tamanho:** 3–5 frases
- **Estrutura:** O que o vídeo mostra → Resultado/aprendizado concreto → Diagnóstico gratuito + link
- **Keywords:** Lean Six Sigma, DMAIC, melhoria de processos, automação com IA, pequenas empresas BH
- **CTA:** Sempre termina com convite ao diagnóstico gratuito

**Exemplo:**
```
Nesse vídeo eu mostro como aplicar a fase Analisar do DMAIC para identificar a causa raiz do retrabalho em processos de atendimento.

Em 4 semanas, a clínica odontológica eliminou o problema que 3 meses de tentativa não resolveram. O método não é novo — o que falta é aplicar direito.

Diagnóstico gratuito de 30 minutos: smartops-ia.com.br/diagnostico-gratuito
WhatsApp: (31) 97203-9180
```

### Tags

Sempre incluir estas tags base + tags de nicho do vídeo:
```
Lean Six Sigma, DMAIC, melhoria de processos, automação com IA,
pequenas empresas, gestão operacional, redução de desperdício,
SmartOps IA, Breno Luiz, Black Belt, BH, Belo Horizonte
```

### Thumbnail Guidelines

- **Background:** Dark (`#06060e` ou `#0d0d1c`)
- **Texto:** Off-white (`#e8e8f0`), bold, máximo 5–6 palavras
- **Accent:** Roxo (`#7c3aed`) para Lean, Verde (`#10b981`) para Automação
- **Screenshot ou visual:** fluxo de processo, dashboard, resultado numérico em destaque
- **Sem:** fundo colorido saturado, fonte muito pequena, muitos elementos
